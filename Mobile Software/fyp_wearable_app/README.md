# Flutter Peer Location System

## Requirements
- Flutter SDK (>= your version)
- Android Studio / VS Code
- Android device or emulator (Real device for location tracking)

## Setup
1. Extract ZIP
2. Run `flutter pub get`
3. Configure Firebase (google-services.json provided)
4. Run `flutter run`

## Notes
- BLE requires a physical Android device
- Location permission must be enabled
- Firebase API keys are included for academic demonstration purposes ONLY.
